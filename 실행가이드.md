# IP Bridge 프로젝트 실행 가이드

## 1단계: 의존성 설치

터미널에서 다음 명령어를 실행하세요:

```bash
npm install
```

## 2단계: 환경변수 설정

프로젝트 루트 디렉토리에 `.env.local` 파일을 생성하고 다음 내용을 입력하세요:

```env
# Supabase 설정 (필수)
NEXT_PUBLIC_SUPABASE_URL=여기에_수퍼베이스_URL_입력
NEXT_PUBLIC_SUPABASE_ANON_KEY=여기에_수퍼베이스_ANON_KEY_입력
SUPABASE_SERVICE_ROLE_KEY=여기에_수퍼베이스_SERVICE_ROLE_KEY_입력

# Anthropic Claude API (필수)
ANTHROPIC_API_KEY=여기에_클로드_API_키_입력

# KIPRIS API (필수)
KIPRIS_API_KEY=여기에_키프리스_API_키_입력

# Toss Payments (선택사항 - 결제 기능 사용 시)
NEXT_PUBLIC_TOSS_CLIENT_KEY=test_ck_D5GePWvyJnrK0W0k6q8gLzN97Eoq
TOSS_SECRET_KEY=test_sk_zXLkKEypNArWmo50nX3lmeaxYG5R

# 앱 URL (선택사항)
NEXT_PUBLIC_APP_URL=http://localhost:3000
```

### API 키 발급 방법

1. **Supabase**: https://supabase.com 에서 프로젝트 생성 후 Settings > API에서 키 확인
2. **Anthropic Claude**: https://console.anthropic.com 에서 API 키 발급
3. **KIPRIS**: https://www.kipris.or.kr 에서 API 키 발급

## 3단계: 데이터베이스 설정

Supabase 대시보드의 SQL Editor에서 `supabase-schema.sql` 파일의 내용을 실행하세요.

## 4단계: 개발 서버 실행

터미널에서 다음 명령어를 실행하세요:

```bash
npm run dev
```

## 5단계: 브라우저에서 확인

서버가 실행되면 다음 주소로 접속하세요:

**http://localhost:3000**

## 문제 해결

### 포트 3000이 이미 사용 중인 경우
다른 포트로 실행하려면:
```bash
npx next dev -p 3001
```

### 의존성 설치 오류
```bash
rm -rf node_modules package-lock.json
npm install
```

### 환경변수 오류
`.env.local` 파일이 제대로 생성되었는지 확인하고, 모든 필수 환경변수가 입력되었는지 확인하세요.
